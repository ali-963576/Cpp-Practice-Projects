Client Management System.


This is a simple Client Management System written in C++ using file handling.
The program allows the user to manage client records such as adding, updating, deleting, searching, and listing clients.
All data is stored persistently in a text file.



هذا مشروع نظام إدارة عملاء بسيط مكتوب بلغة C++ باستخدام التعامل مع الملفات.
يتيح البرنامج إضافة العملاء، تعديل بياناتهم، حذفهم، البحث عنهم، وعرض قائمة العملاء.
يتم حفظ البيانات بشكل دائم داخل ملف نصي

Features.

Add new clients

Display all clients in a formatted table

Update client information

Delete clients

Search for a client by account number

Data persistence using text files
